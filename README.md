# FIFA World Cup 2026 Live Streaming & Tournament Dashboard

Welcome to the official repository for the **FIFA World Cup 2026 Live Streaming Portal & Tournament Dashboard**. This client-side player and data console is optimized for football fans to follow the historic 23rd edition of the tournament hosted across the United States, Mexico, and Canada.

### 🚀 [Launch KRYNN SPORTS Web App](https://krynn-sports.vercel.app/)

---

## 🏆 Key Features

- **Live FIFA Streams**: Decrypted premium streaming feeds supporting DASH (.mpd) and HLS (.m3u8) formats with adaptive bitrate scaling.
- **Dynamic Scoreboard**: Real-time match scores, matchday stats, and possession data synced directly from the SportScore API.
- **Win Probability Engine**: Head-to-head match prediction algorithm showing home win, draw, and away win probability stats.
- **Group Stage Tracker**: Live interactive tables tracking points, goal differences, and stage qualification for all Groups (A to L).
- **Responsive Layout**: Premium glassmorphic interface styled with vanilla CSS, optimized for widescreen desktops, laptops, tablets, and mobile screens.

---

## 📂 Web App Architecture

The dashboard is built on a high-performance stack:
- **Client**: Vanilla HTML5, CSS custom properties (dark cinema theme), and lightweight JavaScript.
- **Player**: Shaka Player integration with built-in ClearKey DRM client.
- **Server**: Express.js static backend server with API proxy caching.

---

## 🔗 How to Connect & View

1. Access the web app: **[https://krynn-sports.vercel.app](https://krynn-sports.vercel.app/)**
2. Select any active feed from the **Live Broadcast Feeds** drawer.
3. Switch between **Feeds**, **Chat**, **Matches**, and **Standings** tabs.
4. Toggle **Theater Mode** or **Picture-in-Picture (PiP)** to multitask while keeping the stream in view.
